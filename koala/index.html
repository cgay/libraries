Koala

Koala is an HTTP server written in Dylan. It includes Dylan Server Pages, a template engine similar to JSP without scriptlets. But because it is written in Dylan, and can take advantage of Dylan's powerful language features, creating web apps in it should be much more concise and straightforward than with JSP.

The HTTP server portion of Koala was written mostly by Gail Zacharias. She moved on to other things before it was in a functioning state, so I finished it up and added Dylan Server Pages.

The current version is v0.1. It is provided mostly for the interest and amusement of other Dylan programmers and is nowhere near production quality. The current version is for Windows only. I plan to fix this as soon as I can by porting to Gwydion Dylan. If you want to help, let me know!

Downloading

Download the current sources only: koala.zip
Requires the Functional Developer IDE to compile and run. Open and run the koala-example.hdp project in koala/sources/example/. Then point your browser at http://localhost:7020/demo/home.dsp.

Download the sources and binaries: koala-0.1b.zip
For anyone who doesn't have Functional Developer installed but would like to see how Koala/DSP works. You can start the demo application by double clicking on koala-example.exe in the koala/sources/example/release/ directory. Then point your browser at http://localhost:7020/demo/home.dsp.

There is a small demo application included, which shows how to use the features of Koala and DSP.
